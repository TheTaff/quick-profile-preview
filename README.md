
# 📇 Quick Profile Preview (DevCard Generator)

Интерактивное веб-приложение для создания стильных карточек-визиток разработчика в реальном времени.

## Особенности
- **Мгновенное превью:** Данные обновляются сразу при вводе в форму.
- **Интеграция с GitHub:** При вводе никнейма автоматически подтягивается аватар пользователя и обновляется ссылка на профиль.
- **Темы оформления:** Поддержка светлого и темного режимов для карточки.
- **Современный дизайн:** Использование эффектов Glassmorphism и неоморфизма в интерфейсе.

## Технологии
- **HTML5** — семантическая разметка.
- **CSS3** — кастомные свойства (переменные), Flexbox, анимации и градиенты.
- **JavaScript (Vanilla JS)** — манипуляции с DOM и обработка событий.
